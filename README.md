## Introduction
This repository is a personal project For OpenWRT. An simple and lightweight Mesh System.

* Maintain clinets DB and easy to get sta's detail
* Network healing
* AIR Monitor, Monitor STA 
* Seemless roaming via 802.11K/V/R
* bandsteering
* Best Topology 
* Auto-Configuration
